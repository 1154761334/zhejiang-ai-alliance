全称为浙江省 AI 智能体产业发展联盟（Zhejiang AI Agent Industry Alliance）。

联盟于 **2026年2月3日** 在杭州正式成立，由浙江省经济和信息化厅指导，汇聚阿里云、西湖大学、科大讯飞、钉钉、中控技术、传化智联等 **200余家** 行业领军单位共同发起。

## 联盟愿景

打造全球领先的 AI 智能体产业创新高地：
- **技术突破**：攻克 Agent 核心算法难题
- **产业应用**：打造智能体+千行百业的标杆应用场景
- **生态繁荣**：联动资本与开发者体系，打造活跃的开源创新社区

## 六大核心方向

| 方向 | 说明 |
|------|------|
| 技术攻关 | 依托西湖大学与领军企业，攻克 Agent 核心算法与框架 |
| 场景赋能 | 深入制造、医疗、金融等行业 |
| 生态构建 | 举办开发者大会与创新大赛 |
| 标准制定 | 参与智能体分级、评测与安全伦理标准 |
| 产融结合 | 联动产业基金与创投机构 |
| 人才培育 | 实施"AI+X"人才培养计划 |

## 发展目标

根据浙江省"十五五"规划：
- **2027年**：全省人工智能核心产业营收突破 **1万亿元**
- **2030年**：达到 **1.2万亿元**，构建芯片、大模型、智能体、应用服务全产业链协同格局

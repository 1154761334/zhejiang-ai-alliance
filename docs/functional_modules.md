# 功能模块说明 (Functional Modules)

> 更新日期: 2026-03-11

## 1. 门户首页与营销模块
- **Landing Page**: 基于联盟定位重绘，Hero 区域引导企业“成员入库”。
- **入会权益**: 三级会员体系 (观察员、普通成员、理事单位) 的分级权益展示。
- **文档中心 (/docs)**: 包含《联盟简介》、《入会指南》，指导企业完成线上入库。

## 2. 新闻动态与资讯管理
- **动态管理**: 秘书处在 Directus `articles` 集合录入内容，前端 `/blog` 自动同步。
- **内容分类**: 联盟动态 (news)、政策解读 (policy)、行业洞察 (insight)。

## 3. 企业能力档案采集系统 (核心)
- **向导式表单**: 位于 `/dashboard`，提供高度沉浸式的填报体验：
  - **步骤 1: 基础画像**: 社会信用代码、营收规模、研发人数等。
  - **步骤 2: 产品矩阵**: 动态添加企业旗下的 Agent 产品、大模型底座能力。
  - **步骤 3: 标杆案例**: 收集已落地的行业案例、量化效果与佐证材料。
  - **步骤 4: 生态需求**: 融资需求、市场对接、算力痛点、技术互补意向。
- **草稿机制**: 每一秒自动同步至 Directus，用户退出后重连可直接回填。

## 4. 成员入库审批系统 (秘书处专用)
- **运营驾驶舱 (/admin)**: 即时统计待审核数量、入库企业总数、最新动态。
- **材料审批台 (/admin/companies)**: 
  - **清单模式**: 支持按状态、成熟度、赛道筛选。
  - **深度评审**: 秘书处在此填写评级意见 (A/B/C)、内部反馈，执行“发布/驳回”操作。

## 5. 账号体系与 RBAC 权限
- **自主申请**: 企业通过 `/register` 提交基本邮箱信息即可创建账号并自动登录。
- **权限隔离**: 
  - **普通企业**: 仅能访问自己的 `/dashboard` 与 `/billing`。
  - **管理员**: 拥有后台全局权限及数据导出能力。

## 6. 会费与入库状态查询
- **状态追踪 (/dashboard/billing)**: 企业实时查看自己的入库进度 (草稿/审核中/正式入库) 以及由秘书处评定的“成熟度等级”。
